Dans cette boutique de vente informatique en ligne, nous avons quatre tables principales :

Utilisateur : contient les informations relatives aux utilisateurs, telles que leur nom, prénom, adresse e-mail, mot de passe, etc.
    => L'utilisateur s'enregistre depuis le site et il est enregistrer dans la base de données.
Produit : contient les informations relatives aux produits, telles que leur nom, description, prix, quantité en stock, etc.
    => Les produits sont ajoutés dans la base de données manuellement par la gestionnaire du site, via le fichier data.sql.
Commande : contient les informations relatives aux commandes passées par les utilisateurs, telles que la date de commande, le montant total, le statut de la commande, etc.
    => L'utilisateur passe par le panier pour completer et finaliser sa commande.
Panier : contient la liste des produits contenu dans le panier d'un utilisateur
    => L'utilisateur passe par la page des categories puis ajoute le produit qu'il souhaite acheter au panier
